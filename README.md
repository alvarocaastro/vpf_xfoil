# Variable Pitch Fan вҖ” Aerodynamic Analysis Pipeline

Python pipeline for the complete aerodynamic analysis of a variable-pitch fan (VPF).
Covers NACA aerofoil selection, XFOIL simulations, compressibility corrections,
blade kinematics with cascade effects, reverse thrust, and SFC reduction estimation.

---

## Reference engine

All geometric and operating parameters replicate a **GE9X-105B1A (Boeing 777X)**:
BPR вүҲ 10, 3.40 m fan, 16 wide-chord composite blades, design RPM 2200.

### Flight conditions

| Condition | M_rel @ mid-span | Va [m/s] | Ncrit |
|-----------|-----------------|----------|-------|
| Takeoff   | 0.85            | 180      | 4.0   |
| Climb     | 0.85            | 155      | 4.0   |
| Cruise    | 0.93            | 150      | 4.0   |
| Descent   | 0.80            | 125      | 4.0   |

Va is the axial velocity at the fan face. M_rel is evaluated at mid-span using the
relative velocity W = вҲҡ(VaВІ + UВІ), which is the physically relevant Mach number for
2D compressibility corrections.

### Blade sections

| Section  | Radius [m] | U [m/s] @ 2200 rpm | c [m] | Пғ    |
|----------|------------|--------------------|-------|------|
| Root     | 0.53       | 122.1              | 0.36  | 1.73 |
| Mid-span | 1.00       | 230.4              | 0.46  | 1.17 |
| Tip      | 1.70       | 391.7              | 0.46  | 0.69 |

## Pipeline architecture

```
run_analysis.py
в”Ӯ
в”ңв”Җв”Җ Stage 1 вҖ” Aerofoil selection
в”Ӯ   в””в”Җв”Җ XFOIL @ Re_cruise, M_cruise вҶ’ CL/CD ranking вҶ’ NACA 65-410
в”Ӯ
в”ңв”Җв”Җ Stage 2 вҖ” Final XFOIL simulations
в”Ӯ   в””в”Җв”Җ 12 polars (4 conditions Г— 3 sections); auto-retry, convergence detection
в”Ӯ
в”ңв”Җв”Җ Stage 3 вҖ” Compressibility corrections
в”Ӯ   в”ңв”Җв”Җ PrandtlвҖ“Glauert: CL_PG = CL / вҲҡ(1 вҲ’ MВІ)
в”Ӯ   в”ңв”Җв”Җ KarmanвҖ“Tsien:    full iterative correction; output column вҶ’ ld_corrected
в”Ӯ   в””в”Җв”Җ Korn (wave):     CD_wave вҲқ (M вҲ’ M_dd)вҒҙ for M > M_dd
в”Ӯ
в”ңв”Җв”Җ Stage 4 вҖ” Performance metrics
в”Ӯ   в””в”Җв”Җ CL/CD_max, Оұ_opt, CL_max, stall margin, О”Оұ VPF vs fixed pitch
в”Ӯ
в”ңв”Җв”Җ Stage 5 вҖ” Pitch kinematics (3D fan analysis)
в”Ӯ   в”ңв”Җв”Җ [A] Cascade: Weinig (K_weinig) + Carter (Оҙ_carter)
в”Ӯ   в”ңв”Җв”Җ [B] 3D rotational correction: Snel (О”CL вҲқ (c/r)ВІ В· CL_2D)
в”Ӯ   в”ңв”Җв”Җ [C] Design twist + off-design penalty with single actuator
в”Ӯ   в””в”Җв”Җ [D] Stage loading: ideal (Оұ_opt_3D) vs real (single actuator) scenario
в”Ӯ
в”ңв”Җв”Җ Stage 6 вҖ” Reverse thrust
в”Ӯ   в”ңв”Җв”Җ Negative pitch sweep О”ОІ вҲҲ [вҲ’25В°, вҲ’5В°] at N1 = 65%
в”Ӯ   в””в”Җв”Җ VPF mechanism weight vs conventional cascade reverser
в”Ӯ
в””в”Җв”Җ Stage 7 вҖ” SFC and mission analysis
    в”ңв”Җв”Җ Оө(r, cond) = (CL/CD)_vpf / (CL/CD)_fixed_ref
    в”ңв”Җв”Җ О”О·_fan = П„ В· (ОөМ„ вҲ’ 1) В· О·_fan,base
    в”ңв”Җв”Җ SFC_new = SFC_base / (1 + О”О·/О·_base)
    в”ңв”Җв”Җ Mission fuel burn per phase, sensitivity to П„
    в””в”Җв”Җ GE9X thermodynamic cycle validation + parametric Cl/Cd sweep
```

**Python 3.10+** and **XFOIL** accessible on the `PATH`.

```bash
python -m venv .venv
# Windows:  .venv\Scripts\activate
# Linux:    source .venv/bin/activate

pip install -r requirements.txt
```

If XFOIL is not on the `PATH`:


## Module dependency rules

No stage imports directly from another stage. Communication is exclusively through
files in `results/` and the `StageNResult` contracts in `pipeline/contracts.py`.

```
config/domain.py    вҶҗ settings.py (constructs dataclasses from YAML)
settings.py         вҶҗ all code (path constants, get_settings())
validation/         вҶҗ adapters, postprocessing, run_analysis.py
pipeline/contracts  вҶҗ run_analysis.py (inter-stage contract validation)
postprocessing/     вҶҗ run_analysis.py, stage orchestrators
```

## Key equations

### Stage 3 вҖ” Compressibility corrections

| Correction | Equation |
|------------|----------|
| PrandtlвҖ“Glauert | `CL_PG = CL / вҲҡ(1 вҲ’ MВІ)` |
| KarmanвҖ“Tsien | `CL_KT = CL_PG / [ОІ + (MВІ/2ОІ) В· CL_PG/2]` |
| Korn wave drag | `M_dd вүҲ Оә/cos(Оӣ) вҲ’ (t/c)/cosВІ(Оӣ) вҲ’ CL/(10cosВі(Оӣ))` вҶ’ `О”CD вҲқ (M вҲ’ M_dd)вҒҙ` |

### Stage 5 вҖ” Cascade & rotational corrections

```
Пғ(r)        = c(r) / s(r)            s(r) = 2ПҖr / Z
K_weinig(Пғ) = (ПҖ/2В·Пғ) / arctan(ПҖВ·Пғ/2)
Оҙ_carter(r) = m В· Оё / вҲҡПғ(r)          m = 0.23 (NACA 6-series)
О”CL_snel(r) = a В· (c/r)ВІ В· CL_2D     a = 3.0
```

### Stage 7 вҖ” SFC model

```
Оө(r, cond)   = (CL/CD)_vpf / (CL/CD)_fixed_ref
О”О·_fan       = П„ В· (ОөМ„ вҲ’ 1) В· О·_fan,base
SFC_new      = SFC_base / (1 + О”О·_fan / О·_fan,base)
```

`П„ вүҲ 0.5` damps the ideal 2D gain to reflect 3D losses not captured in isolated-section analysis.
